# Mental Health Tracker Web App

## Description

This is a web application developed for the Platform Based Programming course at Universitas Indonesia. The app is built using the Django framework and enables users to track their mental health by logging mood entries. It uses Django's built-in templating system with HTML templates for the user interface, providing a seamless experience for mood monitoring and management on the web.

## API Integration

This application serves as the Django backend API. The frontend is rendered using Django templates, and the app can be accessed via web browsers. For any external integrations, the API endpoints are available for communication.
